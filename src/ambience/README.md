# Ambience

Passive Weltwirkung ohne direkte Gameplay-Regelwirkung.

## Struktur

- `visual/`
  - rein dekorative Raumwirkung wie Boden-Overlays
- `narration/`
  - passive Atmosphaere- oder Beobachtungstexte wie Showcase-Ansagen

## Gehört bewusst nicht hierhin

- blockierende oder interaktive Weltobjekte
- Showcases als physische Platzierungsobjekte
- Chests, Traps, Türen, Treppen, Loot, Monster

## Ziel

Ambiente soll getrennt von Dungeon-Layout, Interaktion und Kampfregeln wachsen können.
