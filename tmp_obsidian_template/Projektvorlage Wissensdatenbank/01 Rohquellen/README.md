# Rohquellen

Hier liegen die Originalquellen des Projekts.

## Regeln
- Dateien bleiben inhaltlich unveraendert.
- Dateinamen sollen nachvollziehbar sein.
- Wenn sinnvoll, Datum und Quelle im Dateinamen fuehren.
- Die inhaltliche Einordnung erfolgt nicht hier, sondern ueber Wissensseiten und Quellenbewertungen in `02 Wissen`.
