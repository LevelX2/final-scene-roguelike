The Final Scene - Portable Browser-Version

Start:
1. Entpacken.
2. index.html in einem modernen Browser oeffnen.

Enthalten:
- index.html
- styles.css
- dist/game.bundle.js
- assets/

Hinweise:
- Es ist keine Installation von Node.js notwendig.
- Spielstaende und Einstellungen werden im Browser gespeichert.
- Die optionale OpenAI-Sprachausgabe ueber den lokalen Server ist in dieser Version nicht enthalten.
