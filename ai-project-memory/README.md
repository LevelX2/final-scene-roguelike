# AI Project Memory

Diese Wissensbasis ist das projektbezogene Arbeitsgedächtnis für KI-gestützte Arbeit an `The Final Scene`.

Zweck:

- stabilen Projektkontext über mehrere Sitzungen hinweg erhalten
- Entscheidungen, Risiken, Prozesse und verifizierte Erkenntnisse wiederverwendbar machen
- neue belastbare Erkenntnisse aus Codearbeit und Analysen zurück in den Projektkontext führen

Empfohlener Einstieg:

1. `00 Projektstart.md`
2. `02 Wissen/00 Uebersichten/Index.md`
3. `02 Wissen/Prozesse/Arbeitsworkflow Wissenspflege und Projektanfragen.md`
4. `00 Steuerung/Regeldatei KI-Wissenspflege.md`

Hinweis:

- Rohquellen unter `01 Rohquellen/` bleiben unverändert und dienen als historische Referenz.
- Verdichtetes, gepflegtes Projektwissen liegt unter `02 Wissen/`.
